## Описание проектов

Данные проекты были выполнены в ходе обучения в Яндекс.Практикуме по профессии "Аналитик данных".

| Название проекта | Описание | Используемые библиотеки | 
| :---------------------- | :---------------------- | :---------------------- |
| [Анализ пользовательского поведения в мобильном приложении](a_b_test) | В данном проекте мной были изучены принципы событийной аналитики. Я построила воронку продаж, исследовала путь пользователей до покупки. Проанализировала результаты A/B-теста введения новых шрифтов. Сравнила 2 контрольных группы между собой, убедилась в правильном разделении трафика, а затем сравнила с тестовой группой. Выявлено, что новый шрифт значительно не повлияет на поведение пользователей.| *pandas, matplotlib, seaborn* |
| [Анализ убытков приложения ProcrastinatePRO+](analysis_of_business_indicators) | Проведен анализ данных от ProcrastinatePRO+. Рассчитаны различные метрики, использован когортный анализ: LTV, CAC, Retention rate, DAU, WAU, MAU и т.д. Использованы уже написанные ранее функции расчёта метрик. Сделаны выводы по полученным данным.| *pandas, matplotlib, seaborn, NumPy* |
| [Изучение закономерностей, определяющих успешность игр](online_store) | Выявлены параметры, определяющие успешность игры в разных регионах мира. На основании этого подготовлен отчет для магазина компьютерных игр для планирования рекламных кампаний. Проведена предобработка данных, анализ. Выбран актуальный период для анализа. Составлены портреты пользователей каждого региона. Проверены гипотезы: средние пользовательские рейтинги платформ Xbox One и PC одинаковые; средние пользовательские рейтинги жанров Action и Sports разные. При анализе использовала критерий Стьюдента для независимых выборок.| *pandas, matplotlib, NumPy* |
| [Продажа квартир в Санкт-Петербурге — анализ рынка недвижимости](yandex_flat) | На основе данных сервиса Яндекс.Недвижимость определена рыночная стоимость объектов недвижимости разного типа, типичные параметры квартир, в зависимости от удаленности от центра. Проведена предобработка данных. Добавлены новые данные. Построены гистограммы, боксплоты, диаграммы рассеивания.| *pandas, matplotlib* |
| [Анализ поведения пользователей в мобильном приложении](unnecessary_things) | Датасет содержит данные о событиях, совершенных в мобильном приложении "Ненужные вещи". В нем пользователи продают свои ненужные вещи, размещая их на доске объявлений. Проведен анализ связи целевого события — просмотра контактов — и других действий пользователей. Построена диаграмма Санкея с целью определить наиболее популярные сценарии поведения пользователей, матрица корреляций - чтобы определить связь целевго события с остальным действиями пользователей, а также воронки - для конверсий. Сформулированы и проверены гипотезы. Сформирован дашборд в Tableau. | *pandas, matplotlib, plotly, seaborn* |
